### HowTo: Operate

// TODO

The switch cabinet consists of 3 columns of flip switches. Each column is assigned to work with one of the S210 drives. Each column consists of 4 flip switches.

From top to bottom the functionality is assigned as follows:

- switch 0: activate drive, reset previous state
- switch 1: change rotation direction
- switch 2: increase rotation speed
- switch 3: unassigned
- rotary switch: unassigned

E-Stop button: stops the whole application
